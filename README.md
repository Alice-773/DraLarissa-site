🩺 Dra. Larissa Alencar – Landing Page Profissional

Projeto de landing page institucional desenvolvido para uma estomaterapeuta especialista em tratamento avançado de feridas.

O objetivo do projeto é apresentar serviços, reforçar autoridade profissional e facilitar o contato via WhatsApp.

🚀 Tecnologias Utilizadas

HTML5

CSS3

JavaScript

ScrollReveal.js

Git & GitHub

🎯 Funcionalidades

Layout responsivo (mobile first)

Seção de serviços em grid

Animações suaves com ScrollReveal

Botão de contato direto via WhatsApp

Favicon personalizado

Estrutura organizada em múltiplos arquivos CSS
